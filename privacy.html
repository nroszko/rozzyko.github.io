Privacy Policy for SPCCTools Cisco Caller ID Extension Last updated: September 21, 2024 Introduction We are committed to protecting the privacy and security of personal information. This privacy policy outlines how the SPCCTools Cisco Caller ID Extension (“the Extension”) collects, uses, and manages personal data. This policy applies to all users of the Extension within our Control Center. ________________________________________ 1. Information We Collect The Extension collects and processes the following types of personal information when interacting with the Control Center’s Cisco web application and the centralized Dataverse contact database: • Full Name • First and Last Name • Phone Numbers (Primary and Secondary) • Position/Role Information • Cisco-related Numbers • Local Number Indicators • District Information • Contact Status (Active/Inactive) • User IDs (for audit purposes) • Timestamps (Created On, Modified On) • Ownership Information (Owner, Created By, Modified By) This information is collected to facilitate real-time communication and contact management within the Control Center. ________________________________________ 2. How We Use Your Information The personal information collected via the Extension is used for the following purposes: • To display Caller ID information within the Cisco web application used by Control Room operators. • To enable operators to add, edit, and retrieve contact information. • To store contact details in the centralized Dataverse database for ongoing use within the Control Center. • To ensure that communications are efficient and that the correct contact information is available to Control Room operators in real time. • To cache Caller ID data locally for up to 13 hours to optimize performance and reduce the frequency of data retrieval requests from the Dataverse. ________________________________________ 3. Data Storage and Retention • Dataverse Storage: All contact information is stored securely in the Dataverse, our centralized contact database. • Caching: Caller ID information is cached locally in the browser for up to 13 hours to enhance performance. After this period, data is automatically refreshed or can be manually refreshed by the user. • Data Retention: Contact information remains stored in the Dataverse until it is manually deleted or updated by authorized personnel. ________________________________________ 4. Data Sharing The SPCCTools Cisco Caller ID Extension shares personal information between the following systems our internal network: • Cisco Web Application: Contact information is displayed within the Cisco interface for use by Control Room operators. • Power Automate: Data retrieval, updates, and additions are processed through Power Automate flows to ensure real-time synchronization with the Dataverse. • Dataverse: The centralized contact database where all records are stored securely. No personal data collected by the Extension is shared with third parties outside of our company. ________________________________________ 5. Data Security We take appropriate technical and organizational measures to protect the personal data processed by the Extension, including: • Secure transmission of data via HTTPS. • Use of API keys for authentication to ensure that only authorized requests are processed. • Limiting data access to users within the network. • Regular monitoring and maintenance of Power Automate flows to ensure consistent functionality and data protection. ________________________________________ 6. User Rights As a user, you have the following rights with respect to your personal information: • Access: You have the right to access the contact information stored in the Dataverse and to view any details processed by the Extension. • Correction: If any of the contact information associated with you is incorrect, you can request that it be updated or corrected. • Deletion: You may request the removal of contact information if it is no longer relevant or needed. Requests related to access, correction, or deletion of your personal information can be directed to IT Support or the Control Center Lead. ________________________________________ 7. Changes to This Privacy Policy We may update this privacy policy from time to time to reflect changes in the Extension’s functionality or data processing practices. Any changes will be communicated to users of the extension. ________________________________________ 8.
